# radio-listener

## Development

### Create venv and install dependencies

```shell
python3 -m venv .venv
. ./.venv/bin/activate
python3 -m pip install -r ./src/requirements.txt
```

### Update Dependencies

```shell
python3 -m venv .venv
. ./.venv/bin/activate
python3 -m pip freeze > ./src/requirements.txt

```

## Install PyAudio

```shell
sudo apt-get install python-pyaudio python3-pyaudio
```
